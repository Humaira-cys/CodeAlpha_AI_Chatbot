# AI Chatbot using Python & Flask

## Project Description
This project is a simple AI Chatbot developed using Python and Flask. It provides predefined responses based on user input using an intents.json file. The chatbot has a clean web interface where users can send messages and receive instant replies.

## Features
- Interactive chatbot interface
- Built with Python and Flask
- HTML, CSS and JavaScript frontend
- Predefined responses using intents.json
- Enter key support
- Typing animation
- Responsive and user-friendly interface

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- JSON

## How to Run

1. Install Python.
2. Install Flask:

```
pip install flask
```

3. Run the project:

```
python app.py
```

4. Open your browser and visit:

```
http://127.0.0.1:5000
```
